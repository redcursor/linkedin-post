در سری جدید پست ها، وارد بخث ورژن کنترل می شیم
از نسل ها اول ورژن کنترل تا پیاده سازی یک گیت سرور

به علت کمبود وقت احتمالا مطالب جایی بلاگ نشن

مثل همیشه
ایده
نظر
انتقاد
سوال
زیر پست ها آزاد هست و کامنتی حذف نمیشه ، مگر تبلیغاتی

همچنین در این سری کاری به سنتکس ورژن کنترل ها و گیت نداریم
یعنی قصد آموزش
git add
git commit
git push
git pull
etc

رو نداریم، بشتر مطالی گفته میشه که به
ساختار
معماری
پیاده سازی
و درک درست ورژن کنترل هاست
و اهمیت اون برای تیم و ها و برنامه نویس

همچنین تصاویر استفاده شده اس ول جی هستند
استفاده مجدد از اونها برای آموزش بدون ایراد هست
برای استفاده تجاری هزینه باید پرداخت بشه

بریم برای پست بعدی
